# 📊 Task 02 - Exploratory Data Analysis (EDA)

## 🎯 Objective
To analyze the dataset, clean missing values, and visualize patterns to understand passenger data.

## 🛠️ Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

## 📌 Steps Performed
1. Loaded the dataset
2. Checked column names and structure
3. Handled missing values (Age, Fare)
4. Dropped unnecessary column (Cabin)
5. Performed data visualization for key features
6. Analyzed relationships between variables

## 💡 Key Insights
- Most passengers are between 20–40 years old
- Majority of passengers belong to 3rd class
- Male passengers are more than female passengers
- Fare distribution is highly skewed
- No strong relationship between age and fare
